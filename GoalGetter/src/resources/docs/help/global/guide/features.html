Features

GoalGetter ermöglicht ihnen, schnell eine Entscheidung zu treffen, deren Ergebnis Ihre Vorgaben zu mindestens 100% erfüllt.

Dies unterscheidet Diese Software von anderen Methoden wie z.B. der Function Point Methode, wo nicht die "beste" Alternative, sondern das "kleinste Übel" genommen wird.

Im einzelnen werden folgende Möglichkeiten geboten:

  • Anlegen eines Entscheidungsmodells
  • Laden und Speichern in XML
  • Erstellen eines formatierten, druckbaren Berichts mit bester Alternative, Ergebnismatrix und Handlungen
  • grafische Sensibilitätsanalyse

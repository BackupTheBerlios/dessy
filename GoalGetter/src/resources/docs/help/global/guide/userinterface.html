User Interface

Pull-down Menu
The pull-down menu provides you with a complete list of actions available in the tool.

Toolbar
The tool bar gives you quick access to the most important actions.

Navigator (upper left)
You browse the project components in this panel.

Dynamic Help Topics (lower left)
You can click on a topic to view it. The topics are updated dynamically and follow the selection in the navigator.

Presentation Panel (upper right)
Presents a form for the current navigator component selection.

Dynamic Help Contents (lower right)
Shows the contents for the selected dynamic help topic.

Popup Menus
A popup menu is available in the navigator.
